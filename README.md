# ExcelTrans
*A simple program that put matrix to an array by vertical direction<br>
Eg. <br>
Input:<br>
1 2 3<br>
4 5 6<br>
7 8 9<br>
Output:<br>
1<br>
4<br>
7<br>
2<br>
5<br>
8<br>
3<br>
6<br>
9<br>
The
*The program will operate all the sheet with the same name in the new file.
*Only support *.xlsx formate in current version.
